# DS3231_RPi
C++ helper library for DS3231 RTC Module

# API implemented
- Read Time from RTC
- Set RPi System time to RTC
- Set Alarms 1 and 2 on RTC
- Turn alarms on and off
- Set Rate of Alarms
- Read temperature from RTC
